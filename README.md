# LedMaze




# The components of the project:

                                               

1.  Arduino Uno;

2.  Matrice 1088AS;

3.  Driver MAX7219;
    
4.  LCD;
    
5.  1x resistor 10k ohm for MAX7219 driver;
    
6. 1x resistor 220 ohm for LCD;
    
7. 1x condensator 10 uF;
    
8.  1x condensator 0.1 uF (100 nF);
    
9.  Joystick;
    
10.  Button;
    
11. 3x Breadboard;
    
12.  Arduino USB cable;
    
13.  Jumper wires;
